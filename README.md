# Diabetes Risk Prediction using Machine Learning

## 📌 Project Overview
This project predicts diabetes risk using patient medical and lifestyle data using Machine Learning techniques.

## 🧠 Algorithm Used
- Random Forest Classifier

## 📊 Dataset
- PIMA Indian Diabetes Dataset

## ⚙️ Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## ▶️ How to Run
1. Open notebook.ipynb in Google Colab
2. Run all cells

## 📈 Output
- Accuracy score
- Diabetes risk prediction (High / Low)
